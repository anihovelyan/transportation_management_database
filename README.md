# Transportation Management Database

A PostgreSQL database designed for managing transportation operations.

## Technologies
- PostgreSQL
- SQL
- Python

## Features
- Designed a relational database with 12 interconnected tables.
- Implemented primary keys, foreign keys, and integrity constraints.
- Generated randomized sample data using Python.
- Developed analytical SQL queries for passenger activity, revenue, trip performance, and route analysis.
- Created PL/pgSQL functions and triggers to enforce business rules.
- Added indexes to improve query performance.
